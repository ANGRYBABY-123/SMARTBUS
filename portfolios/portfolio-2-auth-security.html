About Me
Built the complete user authentication system for CommuteSafe, a real-time bus tracking platform. Implemented secure login, logout, registration, and session management with BCrypt password hashing and remember-me cookie tokens. Integrated Google OAuth 2.0 for social sign-in, reCAPTCHA verification to prevent bot registrations, and the AuthFilter middleware that protects every route. Enforced strict role-based access control separating ADMIN, DRIVER, and PASSENGER capabilities.
